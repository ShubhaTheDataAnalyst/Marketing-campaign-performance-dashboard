
# 📊 Marketing Campaign Performance Dashboard

## 📌 Overview
This project analyzes marketing campaign data to evaluate performance, optimize strategies, and support data-driven decision-making. The dashboard provides insights across channels, customer segments, and time periods.

---

## 🎯 Objective
- Measure marketing campaign effectiveness  
- Identify high-performing and underperforming campaigns  
- Analyze customer engagement and conversion behavior  
- Optimize marketing spend using data insights  

---

## 📁 Dataset
The dataset includes:
- Campaign details (Campaign ID, Channel, Type)  
- Customer data (Age, Location, Segment)  
- Performance metrics (Clicks, Impressions, Conversions)  
- Financial data (Cost)  
- Date/Time information  

---

## 🛠 Tools & Technologies
- Microsoft Excel (Data Cleaning)  
- Power BI (Dashboard & Visualization)  

---

## 📊 Key Performance Indicators (KPIs)
- Total Campaign Spend  
- Conversion Rate  
- Click-Through Rate (CTR)  
 
 

---

## 📈 Dashboard Features
- Campaign-wise performance analysis  
- Channel comparison (Email, Social Media, Ads)  
- Time-based trends (Monthly/Weekly)  
- Customer segmentation analysis  
- Interactive filters (Date, Channel, Region)

## 🔍 Key Insights
- Email campaigns show higher conversion rates than social media  
- Paid ads generate higher traffic but lower ROI in some cases  
- Specific customer segments contribute more to revenue  
- Weekend campaigns perform better than weekdays  

---

## 💡 Recommendations
- Increase budget for high-performing campaigns  
- Optimize or pause underperforming campaigns  
- Focus on high-converting customer segments  
- Improve targeting strategies for low ROI channels  

---

## 🚀 How to Use
1. Open the `.pbix` file in Power BI  
2. Use filters to explore campaigns and segments  
3. Analyze KPIs and visualizations  

---

## 📎 Project Outcome
- Improved visibility into campaign performance  
- Data-driven decision-making support  
- Better ROI optimization  

---

## 🙌 Conclusion
This dashboard provides a comprehensive view of marketing performance and helps businesses optimize strategies using data insights.
